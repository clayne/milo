## Algorithms

This file list algorithms supported by the library.

### Codecs

- `base-16`
- `base-64`

### Hashing functions

- `sha-1-160`
- `sha-2-224`
- `sha-2-256`
- `sha-2-384`
- `sha-2-512`
- `sha-2-512-224`
- `sha-2-512-256`

### Message authentication codes

- `hmac-sha-1-160`
- `hmac-sha-2-224`
- `hmac-sha-2-256`
- `hmac-sha-2-384`
- `hmac-sha-2-512`
- `hmac-sha-2-512-224`
- `hmac-sha-2-512-256`
- `poly-1305`

### Key derivation functions

- `hkdf-hmac-sha-1-160`
- `hkdf-hmac-sha-2-224`
- `hkdf-hmac-sha-2-256`
- `hkdf-hmac-sha-2-384`
- `hkdf-hmac-sha-2-512`
- `hkdf-hmac-sha-2-512-224`
- `hkdf-hmac-sha-2-512-256`
- `pbkdf-2-hmac-sha-1-160`
- `pbkdf-2-hmac-sha-2-224`
- `pbkdf-2-hmac-sha-2-256`
- `pbkdf-2-hmac-sha-2-384`
- `pbkdf-2-hmac-sha-2-512`
- `pbkdf-2-hmac-sha-2-512-224`
- `pbkdf-2-hmac-sha-2-512-256`

### Ciphers

- `chacha-20`

### Authenticated encryption with associated data

- `chacha-20-poly-1305`
